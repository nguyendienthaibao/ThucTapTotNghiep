# ThucTapTotNghiep
# Website Tuyển Dụng

## Mô tả
Project xây dựng hệ thống tuyển dụng sử dụng NodeJS và MongoDB.

## Công nghệ sử dụng
- NodeJS
- MongoDB
- Express
- React

## Cài đặt môi trường
- Cài đặt MongoDB
- Cài đặt NodeJS

## Cách chạy project

### Bước 1: Cài thư viện

```bash
npm i
cd client
npm i
cd ../DatabaseSetup
npm i
```

### Bước 2: Setup database

```bash
cd DatabaseSetup
node index
```
(Sau khi chạy xong nhấn Ctrl + C)

### Bước 3: Chạy project

```bash
cd ..
npm run dev
```
